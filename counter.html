How to Use Talker Counter

About

Talker Counter is an app for time management during talks/presentations and discussions. It has two modes:

  • Green Phase: Countdown timer (for the talk)
  • Red Phase: Count-up timer (for discussion). Starts automatically when countdown finishes.

Features

  • Configurable time limits for both phases
  • Large, easy-to-read timer display
  • Visual feedback with color changes
  • Pause and resume timer control
  • Fullscreen mode for presentations
  • Keyboard shortcuts for quick control
